3.8 欠損値を除去する

欠損値を除去する

Stataでは欠損値は.で表されます（数値の場合）。以下では，yearという数値データとcomp_codeという文字列データにおいて欠損値があった場合，その行を削除する例を示しています。drop if 変数 == 欠損値という形です。

コマンド：

drop if comp_code == ""
drop if year == .
